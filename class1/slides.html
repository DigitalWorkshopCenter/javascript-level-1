What is JavaScript?

  • Created by Brendan Eich as "LiveScript" in 1995, which got renamed to "JavaScript"
  • Standardized by the ECMAScript specifications. This class covers ES5 (standardized in 2009)
  • A client-side processing language. A browser reads the code and runs it directly.
  • Interfaces with HTML & CSS.
  • Lets you build dynamic webpages that respond to input from users.

HTML

    Uses tags that open with a < and close with a >

    <p> this creates a paragraph.</p>

    Browsers see the tags and create formatting with them

    Some tags have default styles

HTML - Page Structure

<html> </html>

<head> </head>

<body> </body>

HTML - Containers

<p> </p> - paragraph tag

<div> </div>

<header> </header>

<footer> </footer>

<section> </section>

<article> </article>

HTML - Website Layout

Websites have common layout patterns

<header> At the top of the page </header>

<footer> At the bottom of the page </footer>

<nav> Where the site's links are </nav>

HTML - Formatting

<h1> </h1> - header tag (1 is the largest)

<h6> </h6> - header tag (6 is the smallest)

<ul> </ul> - unordered list (bullets)

<ol> </ol> - ordered list (numbers)

<strong> </strong> - bolding text

<em> </em> - italic text

HTML - Content

<a href="http://google.com">Link to Google</a> - link (anchor) tag

<img src="/images/bunny.jpg"> - image tag

HTML - Dividers

<br> - line break

<hr> - horizontal rule (thematic break)

HTML - all other tags

CSS - Cascading Style Sheet

Structure:
<style>
p {
color: red;
}
</style>

CSS - changing looks

background-color: changes the background color of that element

color: changes the font color of that element

colors can be English names like red or blue, or hex codes like #FF0000 and #0000FF

font-family: changes the font of that element

font-size: changs the size of the font

Classes and IDs

. are used when defining a class .christine

# are used when defining an ID #christine

Classes can be used multiple times, IDs are unique.

CSS - box model

margin: space around an element, outside of border

padding: space around an element, inside of border

border: border around an element

Css - all other tags

There are even more CSS properties than HTML tags! You can find a full list of CSS properties here: https://www.w3schools.com/cssref/default.asp
